Documentation

Learn how to create and share your custom DP templates

Quick Start

Get started with DP Generator in three simple steps:

1

Create a Template

Go to the Create Template page and upload your template image (PNG with transparency recommended). Configure the photo position, size, and text settings.

2

Save & Get Link

Give your template a name and save it. You'll receive a unique shareable link that you can distribute to your users.

3

Share with Users

Users can visit your link, upload their photo, optionally add their name, and generate their personalized DP in seconds.

Creating Templates

Template Image Requirements

  • Format: PNG recommended (supports transparency), JPG also accepted
  • Size: Maximum 16MB file size
  • Dimensions: Square images (1:1 ratio) work best, but any aspect ratio is supported
  • Transparency: Use PNG with transparent areas where the user's photo will appear

Template Settings

Setting Description
Template Name A unique identifier for your template (required)
Event Name Displayed as the page title when users generate their DP
Page Subtitle Secondary text shown below the event name
Footer Text Optional text displayed at the bottom of the generation page
Share Message Pre-filled text when users share their DP to social media

Photo Settings

Setting Description
Photo Shape Choose between circle or rectangle for the user's photo
Photo Size Size of the photo area as a percentage of the template (15-70%)
Photo Position Drag the photo handle to position where users' photos will appear

Text Settings

Setting Description
Font Size Size of the user's name text (2-10% of image height)
Text Color Color of the user's name text
Text Position Drag the text handle to position where names will appear
Custom Font Upload a custom font file (.ttf, .otf, .woff)

Note: The name field is optional for users. If your template doesn't need text, users can leave it blank.

Managing Templates

Viewing Templates

Visit the Generated Templates page to see all your saved templates. You can:

  • Search: Filter templates by name using the search box
  • View Details: Click on a template card to see full configuration
  • Copy Link: Quickly copy the shareable link to your clipboard

Editing Templates

To edit an existing template:

  1. Go to Generated Templates
  2. Click on the template you want to edit
  3. Click the "Edit" button in the modal
  4. Make your changes and save

Editing a template updates it in place - the shareable link remains the same.

Deleting Templates

To delete a template, open its details modal and click "Delete". This action cannot be undone, and the shareable link will no longer work.

User Experience

What Users See

When users visit your template link, they see a simple form with:

  • Your event name and subtitle
  • An optional name input field
  • A photo upload area (supports PNG, JPG, HEIC up to 16MB)
  • A "Generate DP" button

After Generation

Once the DP is generated, users can:

  • Download: Save the image to their device
  • Share: Post directly to Twitter, LinkedIn, or Instagram
  • Create Another: Generate a new DP with different settings

API Reference

DP Generator provides a simple REST API for programmatic access.

Endpoints

GET
/api/get-config?config_id={id}

Retrieve a specific template configuration.

GET
/api/list-configs

List all saved template configurations.

POST
/api/save-config

Create or update a template configuration.

DELETE
/api/delete-config?config_id={id}

Delete a template configuration.

POST
/api/process-image

Generate a DP image. Accepts multipart form data with image and configuration.

Tips & Best Practices

1

Design for the photo area

Leave a clear, appropriately-sized area in your template where users' photos will be placed. Consider the circular crop when designing.

2

Test with different photos

Before sharing, test your template with various photo types (portraits, landscapes) to ensure it looks good in all cases.

3

Keep text readable

Choose a text color that contrasts well with your template background. Avoid placing text over busy areas.

4

Write engaging share messages

Craft a compelling share message that users will want to post. Include relevant hashtags and calls to action.
